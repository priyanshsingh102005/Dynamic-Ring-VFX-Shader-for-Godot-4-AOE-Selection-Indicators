# Dynamic Ring VFX Shader (Godot 4)

A dynamic **ring-based VFX shader** for **Godot 4**, designed for gameplay indicators such as player or enemy selection circles, AOE warnings, targeting zones, danger indicators, and hologram-style UI rings.

This shader is lightweight, highly customizable, and suitable for real-time gameplay.
